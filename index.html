千と千尋の神隠し

El viaje de Chihiro, una película de animación japonesa escrita y dirigida por Hayao Miyazaki y producida por Studio Ghibli.

Tematica

La película se centra en Chihiro, una niña de diez años que, durante un viaje en coche con sus padres, descubre un mundo mágico y misterioso. En este aventura, Chihiro se enfrenta a una serie de pruebas y desafíos que le permiten crecer y madurar. Te encuentras al inicio de la historia, cuando Chihiro y sus padres están buscando su nueva casa en el mundo real. De repente, el carro se descompone en medio de un camino solitario. Para empeorar la situación, animales salvajes comienzan a acechar en los alrededores.

Proceso

Para este proyecto, me inspiré en la película de Miyazaki para crear un videojuego interactivo. Por el videojuego, el jugador se convierte en Chihiro y debe superar diferentes niveles. He desarrollado este pequeño juego utilizando JavaScript orientado a objetos y la biblioteca p5.js. Los recursos gráficos y de audio fueron obtenidos de https://pixel-boy.itch.io/. Para este sitio web, he utilizado el framework Tailwind CSS, que es un framework CSS moderno que ofrece numerosas clases utilitarias. Dependiendo de las clases que se utilizan, genera un archivo CSS que contiene únicamente lo necesario para el sitio, evitando así la carga de datos innecesarios y garantizando una velocidad de carga óptima
